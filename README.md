# Информационный сайт — каркас

Файлы:

- `index.html` — главная страница.
- `css/styles.css` — стили.
- `images/photo.jpg` — поместите сюда вашу фотографию (файл не включён).

Как посмотреть локально:

1. Просто откройте `index.html` в браузере.
2. Или запустите простой HTTP-сервер (рекомендуется для корректной работы некоторых браузеров):

```bash
# Python 3 (в папке c:/Users/MASTER/Desktop/mess)
python -m http.server 8000
# затем открыть http://localhost:8000
```

Замените `images/photo.jpg` на вашу фотографию и отредактируйте текст в `index.html`.
